# 🚀 SmartNotes AI

An AI-powered note assistant that allows users to chat with their notes and retrieve relevant information instantly.

---

## 🧠 Features

- 💬 Chat with your notes  
- 🔍 Keyword-based smart search  
- ⚡ Fast and simple responses  
- 🌐 Full-stack web application  

---

## 🛠 Tech Stack

- **Frontend:** HTML, JavaScript  
- **Backend:** Node.js, Express.js  

---

## 📂 Project Structure

smartnotes-ai/
│── index.js
│── index.html
│── README.md

---

## ▶️ How to Run

1. Install Node.js  
2. Clone the repository  
3. Run the server:
   node index.js  
4. Open in browser:
   http://localhost:3000
